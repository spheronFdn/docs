# Spheron CLI

The Spheron CLI is a Command Line Interface (CLI) tool to create and deploy dapps to web3.

## Installation

To install the Spheron CLI, run the following command:

```
sudo npm install -g @spheron/cli
```

## Available Commands

### Authentication

#### spheron login

**Usage**

```
spheron login
```

---

### Upload

#### spheron upload

**Usage**

```
spheron upload
```

---

### Site

#### spheron init

**Usage**

```
spheron init
```

#### spheron publish

**Usage**

```
spheron publish
```

---

### Template

#### spheron create-dapp

**Usage**

```
spheron create-dapp
```

---

### Organization

#### spheron create-organization

**Usage**

```
spheron create-organization
```
