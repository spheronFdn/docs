# Compute Scaling

## What is compute scaling?

Compute Scaling is the ability to adjust the computational resources allocated to an instance,
aiming to optimize performance and resource utilization.

We offer the following three types of scaling:

1. No Scaling
2. Manual Scaling
3. Auto Scaling

### No Scaling

Pros

- Easy to setup
- Supported in multiple regions
- Totally decenteralized deployed on pallete of akash providers (not only our providers) \*\*\*

Cons

- Cannot scale resources

### Manual Scaling

Pros

- Can scale resources
- Can scale instance count

Cons

- No auto scaling
- deployments are bit more centralized

### Auto Scaling

Pros
